# Welcome to BunGo
BunGo (Bundler 4 Go) is a robust and lightweight Go web framework built for developers who love the simplicity of Go backends, but want the richness of React for their frontend views.

**The core philosophy**: You can write Go server logic alongside React (JSX/TSX) views in the identical repository. No Node.js. No `npm`. No complex build pipelines with Webpack, Vite, or Babel. Everything you need is seamlessly bundled and embedded right into your Go binary!

## When should I use BunGo?
- **Hybrid Applications:** You want a monolithic web app structure without giving up modern frontend tooling.
- **Fast Developer Experience (DX):** You want to avoid maintaining `package.json` files and endless `node_modules`. 
- **Rapid Prototyping and Production:** You want high performance routing and a secure API but don't want to build an entirely separate frontend application.
